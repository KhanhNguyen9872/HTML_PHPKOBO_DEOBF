## üåü Introduction

**HTML PHPKOBO Deobfuscator** is a robust web application designed to help developers and security analysts deobfuscate and analyze HTML code shielded by PHPKOBO obfuscation techniques. It provides a clean, user-friendly interface to transform unreadable code back into its original, structured form.

Built for speed and efficiency, the tool features a powerful code editor, real-time preview, and automatic formatting, making it an essential utility for understanding and debugging protected web content.

## ÔøΩ Getting Started

Follow these steps to set up the project locally.

### Prerequisites
-   **Node.js**: Version 18 or higher recommended.
-   **npm** or **yarn**.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/KhanhNguyen9872/HTML_PHPKOBO_DEOBF.git
    cd HTML_PHPKOBO_DEOBF
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to `http://localhost:5173` (or the port shown in your terminal).

## ÔøΩ Usage

1.  **Paste Code**: Copy the obfuscated HTML/PHPKOBO code into the input editor pane.
2.  **Deobfuscate**: Click the **Deobfuscate** button to process the code.
3.  **Review**: View the cleaned code in the output editor.
4.  **Format**: Toggle formatting options to beautify the structure.
5.  **Preview**: Switch to the **Preview** tab to see how the HTML renders in a browser.
